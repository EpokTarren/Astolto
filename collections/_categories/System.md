# System
Provides information about the bot.


### Commands
- [About](#about)
- [ChangeLog](#changelog)


## About

### Description

Sends a rundown explaining what this bot is.

### Examples
```
a;about
```

---

## ChangeLog

### Description

Display changes from the most recent version

### Aliases
- Changes
- Changed


### Examples
```
a;changelog
a;changed v1.2.1
```

### Arguments

#### version(Optional)

Type: String

Description: The version you want the changelog of.
